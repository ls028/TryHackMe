## Key Concepts Learned

- **SOC Tiers:** L1 triages alerts, L2 investigates, L3 threat hunts  
- **SIEM:** Aggregates logs from all sources, correlates events, fires alerts — Splunk is the industry standard  
- **EDR:** Monitors endpoints for suspicious behavior, can isolate infected machines  
- **Pyramid of Pain:** Hash indicators are easiest for attackers to change. TTPs (Tactics, Techniques, Procedures) are hardest — targeting TTPs hurts attackers the most  
- **Cyber Kill Chain:** Understanding which kill chain stage an attack is in determines the response  
- **Phishing:** Header analysis reveals spoofed senders, email artifact extraction feeds into threat intel
